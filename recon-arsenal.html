🐉 Recon Arsenal

Har din padhne wali file — website, tool, link, example sab ek jagah

1. Whois Lookup DOMAIN INFO

whois.domaintools.com
Example: domain owner, registrar, creation date, name servers dhundhne ke liye
whois.com
Alternative — simpler UI, same data

2. Technology Discovery TECH STACK

sitereport.netcraft.com
Server type, hosting history, tech stack ka overview
Wappalyzer — browser extension
Website open karte hi CMS, framework, JS libraries turant dikhata hai
BuiltWith
Detailed tech report — analytics tools, CDN, hosting provider bhi batata hai

3. Gathering DNS Information DNS

robtex.com
A, MX, TXT, NS records ek jagah dikhata hai
dnsdumpster.com
Visual DNS map banata hai, subdomains bhi kabhi kabhi saath mein dikha deta hai
Terminal: dig target.com ANY ya nslookup target.com

4. Same Server Domains (Reverse DNS) REVERSE DNS

robtex.com
IP daalo, us server pe hosted saare domains dikh jaate hain
Bing search: ip:192.xxx.xx.xx
Same IP pe hosted saari sites Bing list kar deta hai

5. Discovering Subdomains SUBDOMAINS

git clone https://github.com/guelfoweb/knock
Wordlist-based subdomain bruteforce tool
subfinder -d target.com
Fast, passive subdomain enumeration (multiple sources se ek sath)
crt.sh
SSL certificate transparency logs se subdomains milte hain — bahut reliable
amass enum -d target.com
Deep, thorough subdomain + asset discovery (thoda slow but comprehensive)

6. Discovering Sensitive Files FILE DISCOVERY

dirb https://target.com
Default wordlist se hidden directories/files dhundhta hai
gobuster dir -u https://target.com -w wordlist.txt
Faster alternative to dirb, zyada customizable
ffuf -u https://target.com/FUZZ -w wordlist.txt
Sabse fast, filtering options bhi strong hain (status code, size based)

7. Old/Archived URLs HISTORY

web.archive.org (Wayback Machine)
Purani versions of the site dekh sakte ho — kabhi purane API endpoints yahin dikh jaate hain
waybackurls target.com
Terminal tool — Wayback Machine ke saare archived URLs ek list mein nikal deta hai

8. Exposed Services / Infra EXPOSURE

shodan.io
Exposed servers, open ports, IoT devices, kabhi kabhi login panels bhi dikh jaate hain
censys.io
Shodan jaisa hi, thoda alag data set, dono cross-check karne mein useful

9. GitHub Recon CODE LEAKS

site:github.com "target.com" api_key
Google dork — company ke leaked API keys, hardcoded endpoints milte hain
GitHub native search
Company naam + "password", "secret", "config" jaise keywords se search karo

10. Email / Employee Recon OSINT

theHarvester -d target.com -b all
Emails, subdomains, employee names ek sath collect karta hai (multiple sources se)
hunter.io
Company ka email pattern (firstname.lastname@target.com) confirm karta hai

11. Port & Service Scanning NETWORK

nmap -sV -p- target.com
Saare open ports aur unpe chal rahi services identify karta hai
nmap --script vuln target.com
Known vulnerabilities ke against services ko scan karta hai

12. Google Dorking SEARCH RECON

site:target.com filetype:pdf
Public documents dhundhta hai jo internal info leak kar sakte hain
site:target.com inurl:admin
Admin panels ya login pages dhundhne ke liye
site:target.com intitle:"index of"
Open directory listings dhundhta hai — kabhi kabhi poora backup folder khula milta hai

Naya tool seekhoge to isi file mein add ho jayega — ek hi jagah sab recon arsenal.
